Estructura básica de HTML5

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Todo documento <HTML> tiene una estructura básica que asegura que el contenido se presente de manera correcta en los navegadores web. A continuación, se describen los componentes esenciales de un documento <HTML>.


Existen 4 tipos de etiqueta para html 5.


  • < !DOCTYPE > "Declara el tipo de documento."
  • < HTML > "Envoltorio principal del contenido."
  • < HEAD > "Contiene metadatos y recursos."
  • < BODY > "Contiene el contenido visible de la página."

La estructura básica de HTML es fundamental para crear páginas web bien formadas y compatibles con todos los navegadores. Conocer y dominar esta estructura es el primer paso para convertirse en un desarrollador web competente


Estructura HTML


<!DOCTYPE html>
<html lang="es">
<head>
     <meta charset="UTF-8">
     <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <title>Document</title>
</head>
<body>
</body>
</html>



Aclaración de cada etiquetas usadas

<!DOCTYPE html>:
Esta declaración de tipo de documento informa al navegador que el documento está escrito en HTML5. Es esencial para asegurar que el navegador interprete correctamente el código HTML.

<html lang="es">:
La etiqueta <HTML> es el contenedor raíz de todo el contenido de la página web. El atributo lang="es" especifica que el contenido del documento está en español.

<head>:
La etiqueta <HEAD> contiene metadatos y otras configuraciones para el documento HTML, como el título, enlaces a hojas de estilo y scripts. Estos elementos no se muestran directamente en la página web, pero proporcionan información importante.

<meta charset="UTF-8">:
Esta etiqueta <meta> define la codificación de caracteres del documento, especificando que el documento usa la codificación UTF-8. Esto es importante para asegurar que los caracteres se muestren correctamente, especialmente para idiomas con caracteres especiales.

<meta name="viewport" content="width=device-width, initial-scale=1.0">:
Esta etiqueta <meta> es crucial para el diseño responsive. Configura la ventana gráfica del navegador para que el ancho coincida con el del dispositivo del usuario y establece una escala inicial del 100%. Esto ayuda a que la página se vea bien en dispositivos móviles.

<title>:
La etiqueta <title> define el título del documento, que aparece en la barra de título del navegador o en la pestaña. Este título no se muestra en el contenido de la página, pero es importante para el SEO y la usabilidad.

</head>:
Esta es la etiqueta de cierre para el elemento <head>, indicando el fin de la sección de metadatos.

<body>:
La etiqueta <body> contiene todo el contenido visible de la página web, como texto, imágenes, enlaces, etc. Todo lo que se muestra en el navegador se coloca dentro del elemento <body>.

</body>:
Esta es la etiqueta de cierre para el elemento <body>, indicando el fin del contenido visible de la página.

</html>:
Esta es la etiqueta de cierre para el elemento <html>, indicando el fin del documento HTML.
